## 📂 Proyectos  

### 📊 Dashboard de Reclutamiento en Excel (próximamente)
Proyecto práctico enfocado en demostrar habilidades en procesos de selección y análisis de datos.

Incluye:  
- Dataset realista con errores (dirty data)  
- Limpieza y validación de datos  
- Tablas estructuradas  
- Cruce de información (BUSCARX / INDEX-MATCH)  
- Dashboard con KPIs:  
  - Funnel de conversión  
  - Time to hire

👉 Demuestra cómo el uso de datos mejora la organización y toma de decisiones en reclutamiento.
